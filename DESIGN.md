# Otovo Design System

This project uses the design system from `DESIGN (1).md`. Key reference:

## Semantic Tailwind Classes (prefer these)

### Text
- `text-text-primary` — headlines, labels
- `text-text-secondary` — supporting text
- `text-text-tertiary` — hints, placeholders
- `text-text-white` — on dark surfaces
- `text-text-inverse` — on colored backgrounds

### Background
- `bg-bg-primary` — main canvas
- `bg-bg-secondary` — panels
- `bg-bg-tertiary` — hover/striping
- `bg-bg-surface` — cards, drawers

### Buttons
- `_btn _btn-primary` — primary actions
- `_btn _btn-secondary` — secondary
- `_btn _btn-tertiary` — tertiary

### Forms
- `_input` — form inputs
- `_label` — form labels

## Rules
- No raw hex colors in templates
- No inline CSS (except background-image URLs)
- No Bootstrap classes
- Use Poppins font (weights 400, 500, 600)
